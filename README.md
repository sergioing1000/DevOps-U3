# DevOps Laboratory - UNIDAD 3

## Description

This project demonstrates a complete DevOps pipeline developed from scratch using React.

## Technologies

- React 19
- Vite 8
- Vitest
- GitHub Actions
- Jenkins
- Docker
- Kubernetes
- SonarQube
- Snyk
- Prometheus
- Grafana

## Installation

```bash
npm install
```

## Run

```bash
npm run dev
```

## Run Tests

```bash
npm test
```

## Build

```bash
npm run build
```
