# DevOps React Laboratory

## Description

This project demonstrates a complete DevOps pipeline using:

- React + Vite
- GitHub Actions
- Jenkins
- SonarQube
- Snyk
- Docker
- Kubernetes
- Prometheus
- Grafana

## Run locally

```bash
npm install
npm run dev
```

## Run tests

```bash
npm test
```

## Build

```bash
npm run build
```
